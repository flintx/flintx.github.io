2018京东校招题 求幂

发表于 2017-09-17 | 分类于 校招 | | 阅读次数

题意

给定 $n$ ，求有多少个四元组 $a,b,c,d$ 满足 $a^b = c^d,1 \leq a,b,c,d \leq n, n \leq 10^5$

思路

$a^b = c^d$ 等式成立分两种情况：

  • $a^b = a^b$ ，即自身等于自身；
  • $(i^x)^c = (i^y)^d,xc=yd\leq n$ .

关于第一种情况，显然存在 $n^2$ 个这样的等式，因为 $a, b$ 均有 $n$ 种情况。

关于第二种情况，可以先枚举 $(x,y)$ 二元组，当 $(x,y)$ 确定时，二元组 $(c,d)$ 的就只有

种可能情况。
